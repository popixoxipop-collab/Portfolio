# 대표작 — 컵 비율 측정기 (2024-1 메카트로닉스)

## 핵심 포인트

- **센서 융합**: HC-SR04 초음파(액면까지 거리) + CdS 조도(컵 감지 트리거)로 비율을 산출.
- **레지스터 직접 제어**: Timer1 캡처로 초음파 왕복 시간 측정, ADC로 조도 읽기, 4-bit 모드 CLCD 구동.
- **이벤트 제어**: 외부 인터럽트(INT4)로 연속 측정 / 홀드 상태 전환.
- 발표자료·시연영상·완성코드·사진이 한 폴더에 있어 **포트폴리오 제출용으로 바로 활용** 가능.
