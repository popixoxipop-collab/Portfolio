# 대표작 — CNC 3040 설계·부품 (2025-2)

CNC 3040 Pro Max 메탈 프레임 CNC의 최종 어셈블리와 부품 선정·구매 정리 문서를 모은 대표 묶음입니다.

## 대표 파일

| 파일 | 종류 | 내용 |
|------|------|------|
| `CNC_최종어셈블리.SLDASM` | CAD (SolidWorks) | 프레임·리니어 가이드·스핀들 포함 전체 어셈블리 (Git LFS) |
| `CNC 제작용 부품 목록.pdf`, `... 2.pdf` | 문서 | 제작용 부품 목록 |
| `CNC_부품_전체_출처_완성본.xlsx` | BOM | 전체 부품 + 구매 출처 정리 |

## 핵심 포인트

- 리니어 가이드(LGS35) · 사다리꼴 스크류(T8/T12) · NEMA17 스테퍼 · 500W 스핀들(ER11)로 구성한 **메탈 프레임 CNC** 설계.
- 설계(어셈블리) + 부품 선정(BOM) + 구매 출처를 함께 제시해 **설계→구매 검토까지의 완결성**을 보여줌.
- 절삭 하중에 대한 프레임 강성은 SolidWorks Simulation 정적 해석으로 검토(상위 폴더 `Craft-정적 해석 1.*`).
