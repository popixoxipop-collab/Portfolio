## 1. DIY CNC 시스템 구성

### 부품 목록

| 구분 | 부품명 | 역할 |
|------|--------|------|
| 구동 | AXIYAMA AK39H/12-1.8 NEMA17 스테퍼 모터 | X·Y·Z 축 이송 |
| 가이드 | 리니어 가이드 레일 (SBR/MGN 계열) | 정밀 직선 운동 |
| 전달 | 볼스크류 or 타이밍 벨트 | 모터 회전 → 직선 이송 |
| 스핀들 | 에어쿨 스핀들 (775/800W) | 공구 회전 절삭 |
| 제어 | Arduino + GRBL / CNC Shield | G-code 해석 및 모터 구동 |
| 드라이버 | A4988 / DRV8825 스테퍼 드라이버 | 마이크로스텝 제어 |
| 프레임 | 알루미늄 압출재 (20×20, 20×40) | 기계 구조 |

### 부족 항목 (GPT 분석 결과)

| 누락 항목 | 이유 | 권장 사양 |
|-----------|------|-----------|
| 스핀들 마운트 브라켓 | 리니어 레일에 스핀들 직접 고정 불가 | Z축 플레이트 + ER11 척 |
| 커플링 | 스테퍼 축 ↔ 볼스크류 연결 | 5mm-8mm 플렉시블 커플링 |
| 엔드스톱 스위치 | 원점 복귀(Homing) 필수 | 마이크로 리밋 스위치 ×3 |
| 전원공급장치 | 24V/10A 이상 | 스위칭 SMPS |

### CATIA 설계 파일 구조

```
DIY_CNC.CATProduct
├── Frame.CATPart          ← 알루미늄 압출 프레임
├── X_Axis.CATPart         ← X축 리니어 가이드 + 캐리지
├── Y_Axis.CATPart         ← Y축 어셈블리
├── Z_Axis.CATPart         ← Z축 + 스핀들 마운트
├── Stepper_NEMA17.CATPart ← 스테퍼 모터 (다운로드 모델)
└── Spindle.CATPart        ← 스핀들 유닛
```

### GRBL 기본 파라미터

```
$100 = 800   ; X축 steps/mm (NEMA17 + A4988 1/16 마이크로스텝)
$101 = 800   ; Y축 steps/mm
$102 = 800   ; Z축 steps/mm
$110 = 500   ; X 최대 이송속도 [mm/min]
$120 = 10    ; X 가속도 [mm/s²]
$130 = 300   ; X 작업 범위 [mm]
```

## 2. 기계공작실 도면 분석

### 조건

- 실면적: **465 m²** (1층 101호 우측 하단부)
- 도면 단위: mm
- 건물 격자: 9,000mm × 8,000mm 기준

### 치수 추론 과정

```
면적 = 가로 × 세로 = 465 m²

도면 격자 기준 (mm 단위):
  기둥 간격: 9,000mm (가로), 8,000mm (세로) 표준

가능한 조합:
  A = 25,000mm × 18,600mm = 465 m²
  B = 30,000mm × 15,500mm = 465 m²
  C = 27,000mm × 17,222mm ≈ 465 m² (비표준)

도면 격자와 정합:
  9,000 × 3span = 27,000mm (가로)
  8,000 × 2span = 16,000mm ≈ 17,222 (오차 존재)
  → 실제 도면의 치수선 직접 확인 필요
```

### 도면 판독 절차

1. 도면 우측 상단 표제란 → 축척 확인 (예: 1/400)
2. 도면 하단 중앙 치수선 → 실제 mm 값 판독
3. `실제 길이 = 도면 길이 × 축척 분모`
4. 기둥 간격(span)으로 격자 구획 수 확인
5. 면적 = 확인된 가로 × 세로로 검증
