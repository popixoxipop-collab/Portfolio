# 14. AI 에이전트 — AI 강사 Agent

> **한 줄 소개(문제 중심)**: 강의 콘텐츠 제작의 반복 노동을 줄이기 위해, 슬라이드만 입력하면 강의 스크립트→음성→영상을 자동 생성하는 LLM Agent 파이프라인을 구축한 프로젝트.

### 📌 프로젝트 개요 (강의 템플릿)
| 항목 | 내용 |
|------|------|
| 문제 배경 | 강의 영상 제작은 스크립트·녹음·편집까지 시간·반복 비용이 큼 |
| 해결 목표 | 슬라이드 → 강의 영상 생성을 단계형 Agent로 자동화 |
| 기간 / 형태 | KT 에이블스쿨 미니프로젝트 2차 |
| 역할·기여 | Agent 모듈 구현(정보분해·스크립트·검토·TTS·영상·그래프) |
| 기술 스택 | Python, LLM, Agent 그래프, TTS, 영상 합성, gradio |
| 기술 선택 이유 | 다단계 의존 작업이라 **Agent 그래프**로 단계 분리·재실행이 용이 / **gradio**로 빠른 웹 데모 |

### 🧩 문제 → 영향 → 해결 → 결과
- **문제**: 슬라이드 정보만으로 자연스러운 강의 흐름·음성·영상 생성
- **영향**: 맥락 누락 시 강의 품질 저하, 단계 결합 시 디버깅 난이도 상승
- **해결**: 정보분해 → 페이지 내용 생성(검색 보강) → 스크립트 → 내용 검토 → TTS → 슬라이드+음성 영상 합성 → 전체 Agent 그래프 연결
- **결과**: 슬라이드 입력 → 강의 스크립트·음성·영상 산출 파이프라인 완성, gradio 데모로 시연

### 💡 배운 점 · 향후 개선
- **배운 점**: Agent 그래프 설계, '검토 노드'로 품질 보정
- **향후 개선**: 단계별 비용·지연 모니터링, 음성·영상 품질 평가 지표화

## 핵심 구성 (Agent 파이프라인)

| 단계 | 내용 |
|------|------|
| 입력 프롬프트 | 강의 목소리·톤·스타일 지침 설정 |
| 정보 분해 | 슬라이드 저장·관리, 불필요 정보 제거, 제목 추출 |
| 페이지별 내용 생성 | 슬라이드 정보 + 검색 기반 부연 설명 생성 |
| 강의 스크립트 생성 | 전체 흐름을 구상해 페이지별 스크립트 작성 |
| 내용 검토 | 페이지 내용과 스크립트 정합성 검토 |
| 음성 변환(TTS) | 지정한 목소리·톤으로 음성 합성 |
| 영상 제작 | 슬라이드 스냅샷 + 음성 합성 → 페이지/전체 강의 영상 |
| 웹 연결 | **gradio** UI로 음성 변환·실행 |
| Agent 그래프 | 전체 단계를 하나의 Agent 그래프로 연결 |

## 기술 요소

- LLM 프롬프트 엔지니어링, 멀티 스텝 **Agent 그래프** 구성
- 검색(보강) → 스크립트 생성 → 검토의 단계적 파이프라인
- **TTS 음성 합성**, 슬라이드-음성 **영상 합성**
- **gradio** 기반 웹 데모
