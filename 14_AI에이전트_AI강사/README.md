# 14. AI 에이전트 — AI 강사 Agent

> **KT 에이블스쿨(AIVLE) 미니프로젝트 2차.** 슬라이드 자료를 입력하면 **강의 스크립트를 생성하고 음성·영상으로 변환**하는 LLM 기반 "AI 강사 Agent"를 구축한 프로젝트입니다.

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 주제 | AI 강사 Agent v2.0 — 슬라이드 → 강의 영상 자동 생성 |
| 형태 | LLM Agent 파이프라인 (멀티 노드 그래프) |
| 산출물 | Jupyter 노트북(`AI_06반_17조.ipynb`), PDF 보고서 |
| 플랫폼 | KT AIVLE School |

## 핵심 구성 (Agent 파이프라인)

| 단계 | 내용 |
|------|------|
| 입력 프롬프트 | 강의 목소리·톤·스타일 지침 설정 |
| 정보 분해 | 슬라이드 저장·관리, 불필요 정보 제거, 제목 추출 |
| 페이지별 내용 생성 | 슬라이드 정보 + 검색 기반 부연 설명 생성 |
| 강의 스크립트 생성 | 전체 흐름을 구상해 페이지별 스크립트 작성 |
| 내용 검토 | 페이지 내용과 스크립트 정합성 검토 |
| 음성 변환(TTS) | 지정한 목소리·톤으로 음성 합성 |
| 영상 제작 | 슬라이드 스냅샷 + 음성 합성 → 페이지/전체 강의 영상 |
| 웹 연결 | **gradio** UI로 음성 변환·실행 |
| Agent 그래프 | 전체 단계를 하나의 Agent 그래프로 연결 |

## 기술 요소

- LLM 프롬프트 엔지니어링, 멀티 스텝 **Agent 그래프** 구성
- 검색(보강) → 스크립트 생성 → 검토의 단계적 파이프라인
- **TTS 음성 합성**, 슬라이드-음성 **영상 합성**
- **gradio** 기반 웹 데모
