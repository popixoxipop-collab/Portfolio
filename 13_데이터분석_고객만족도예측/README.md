# 13. 데이터 분석 — 고객 만족도 예측

> **KT 에이블스쿨(AIVLE) 미니프로젝트 1차.** "고객 만족도 예측을 통한 서비스 개선"을 주제로 한 데이터 분석 팀 프로젝트입니다. 데이터 전처리부터 모델링·해석까지(미션 1~8)를 수행하고 발표로 마무리했습니다.

- **데이터 이해·전처리**: 결측치·이상치 처리, 피처 정리
- **탐색적 분석(EDA)**: 만족도와 주요 변수 간 관계 시각화

- **해석·제언**: 분석 결과를 서비스 개선 관점으로 정리해 발표
