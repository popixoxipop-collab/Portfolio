# 09. 기계진동학 — 강제·지지 진동 응답 분석

> 1자유도(SDOF) 진동계의 **강제 진동에 의한 변위 응답**과 **지지(기초) 가진에 의한 힘 전달률**을 MATLAB으로 파라미터 스터디한 과제입니다. 감쇠계수 c, 강성 k, 질량 m을 각각 변화시키며 주파수 응답을 비교합니다.

---

## 문제 설정

차량 서스펜션을 모사한 1자유도계.

| 파라미터 | 값 | 의미 |
|----------|-----|------|
| F₀ | 3675 N | 가진력 진폭 |
| m | 1500 kg | 질량 |
| c | 2000 N·s/m | 감쇠계수 |
| k | 183,750 N/m | 스프링 강성 |
| Y | 0.05 m | (지지 가진) 기초 변위 진폭 |

이때 고유진동수 `ωₙ = √(k/m) = √122.5 ≈ 11.07 rad/s`, 감쇠비 `ζ = c / (2√(mk))`.

---

## Problem 1 — 변위 응답 |X(ω)|

강제 진동의 정규화 변위 응답:

```
X(ω) = δ / √[ (1 − (ω/ωₙ)²)² + (2ζ·ω/ωₙ)² ] ,   δ = F₀/k
```

```matlab
delta  = F0 / k;
zeta   = c / (2 * sqrt(m * k));
nomega = sqrt(k / m);
X = delta ./ sqrt((1 - (omega/nomega).^2).^2 + (2*zeta*(omega/nomega)).^2);
```

- **c 변화** (100·1000·2000·5000): 감쇠가 작을수록 공진 피크가 급격히 커짐.
- **k 변화** (5e3~5e4): 강성이 클수록 고유진동수가 높아지고 정적 변위 δ는 작아짐.
- **m 변화** (1000~5000): 질량이 클수록 고유진동수가 낮아져 피크 위치가 이동.

## Problem 2 — 힘 전달률 F_T

지지(기초) 가진 시 질량으로 전달되는 힘:

```
F_T = k·Y·(ω/ωₙ)² · √[ (1 + (2ζ·ω/ωₙ)²) / ((1 − (ω/ωₙ)²)² + (2ζ·ω/ωₙ)²) ]
```

```matlab
F_T = k.*Y.*((omega/nomega).^2).*sqrt( ...
        (1 + (2*zeta*(omega/nomega)).^2) ./ ...
        ((1 - (omega/nomega).^2).^2 + (2*zeta*(omega/nomega)).^2) );
```

→ c·k·m을 각각 변화시키며 전달률 곡선을 비교, **진동 절연(vibration isolation)** 설계 관점에서 해석.

---

## 배운 점

- 공진 부근에서 **감쇠비 ζ**가 응답 피크를 결정하는 핵심 인자임을 그래프로 확인.
- 변위 응답과 힘 전달률의 **주파수 응답 함수(FRF)** 형태 차이를 이해.
- 진동 절연 설계에서 `ω/ωₙ > √2` 영역이 절연 구간임을 응답 곡선으로 검증.
