## 핵심 포인트

- **열전달 실험**: 정상상태 온도 구배로 `Q = −k·A·(dT/dx)` 에서 열전도도 산출.
- **엘리베이터 제어**: 두 외부 인터럽트로 상승/하강/정지/방향전환을 처리하는 상태머신, 스위치 디바운싱 적용.
- 실험 보고서 + 제어 코드 + 작업 스크린샷으로 **실험 분석과 임베디드 구현 경험을 동시에** 제시.
