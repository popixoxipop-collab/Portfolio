# 12. 전공진로설계

> 기계공학 전공 진로 탐색 및 설계 활동을 정리한 과목 자료입니다. 전공 역량 점검과 진로 계획 수립을 위한 활동 문서(별표 양식)를 포함합니다.

---

## 포함 자료

| 파일 | 내용 |
|------|------|
| `원본/전진설/【별표 20】 전공진로설계 활동.hwpx` | 전공진로설계 활동지 |
| `원본/전진설/【별표 25】 전공진로설계 활동.hwpx` | 전공진로설계 활동지 |
| `원본/전진설/【별표 29】 전공진로설계 활동.hwpx` | 전공진로설계 활동지 |

> `.hwpx`는 한글(HWP) 문서 형식입니다. 한컴오피스 또는 호환 뷰어로 열람할 수 있습니다.

---

## 비고

이 분야는 설계·해석 프로젝트와 달리 진로 설계 활동 기록으로, 포트폴리오 전반의 학습 동기와 방향성을 보여주는 보조 자료입니다.
