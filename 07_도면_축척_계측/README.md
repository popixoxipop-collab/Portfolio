## 핵심 개념 정리

### 1. 축척 (Scale)

```
도면 축척 표기: 1 : N  (예: 1/400 → 도면 1mm = 실제 400mm)

실제 길이 = 도면상 길이 × N
도면 길이  = 실제 길이  / N
```

#### 주요 축척 기준표

| 축척 | 용도 |
|------|------|
| 1/1 | 원척 (실물 크기) |
| 1/50 | 단면 상세도 |
| 1/100 | 평면도, 입면도 |
| 1/200 | 소규모 건물 배치도 |
| 1/400 | 대형 건물 평면도 ← 본 도면 |
| 1/1000 | 단지 배치도 |

#### 계산 예시

```
도면 정보: 축척 1/400, 도면 치수선 = 58,000mm

실제 가로 길이:
  58,000mm ÷ 400 = 145m   ← 축척 환산 불필요
  (도면 치수선은 이미 실제 치수를 mm로 표기)

주의: 도면 치수선 값은 이미 실제 치수 [mm]
     축척은 도면 용지 위 물리적 길이를 측정할 때만 사용
```

### 2. 외벽 면적 계산

#### 냉난방 부하 계산용 외벽 면적 데이터 (도면 기준)

| 방향 | 구분 | 수량 | 폭 [mm] | 높이 [mm] | 면적 [m²] | 비고 |
|------|------|------|---------|----------|----------|------|
| 동측 외벽 | 벽체 | 1 | 80,400 | 3,500 | 116.6 | 총 외벽 |
| 동측 | 창문 | 16 | 4,200 | 2,000 | −134.4 | 개구부 (-) |
| 동측 | 문 | 1 | 2,600 | 2,000 | −5.2 | 개구부 (-) |
| 동측 | 창문2 | 1 | 7,200 | 3,500 | −25.2 | 개구부 (-) |
| 서측 외벽 | 벽체 | 1 | 6,000 | 3,500 | 48.3 | |
| 서측 | 창문 | 1 | 7,800 | 3,500 | 27.3 | |
| 남측 외벽 | 벽체 | 1 | 35,200 | 3,500 | 116.6 | |
| 남측 | 창문 | 1 | 35,200 | 3,500 | 123.2 | |

#### 순 외벽 면적 계산

```
순 면적 = 전체 벽면적 - 개구부 면적

동측:
  116.6 - 134.4 - 5.2 - 25.2 = -48.2 m²  ← 음수: 개구부 과다, 재확인 필요

공식:
  A_net = Σ(벽면적) - Σ(창·문 개구부 면적)
  단위 확인: mm × mm → m² 변환 = /1,000,000
```

### 3. 도면 치수 읽기 절차

```
1. 표제란 확인
   └─ 도면명, 축척, 도면번호, 작성일

2. 주 치수선 확인
   └─ 전체 치수 (건물 외곽)
   └─ 부 치수 (개구부, 기둥 간격)

3. 기둥 간격(Span) 파악
   └─ 모듈 격자: 보통 3,000 / 6,000 / 9,000 mm

4. 면적 검증
   └─ 가로 × 세로 = 도면에 표기된 실면적과 일치 여부 확인

5. 단위 통일
   └─ 도면: mm 단위
   └─ 면적: m² 단위 (÷ 1,000,000)
   └─ 냉방부하 계산: m² 사용
```

### 4. 축척과 기계공학 연계

| 도면 종류 | 축척 예시 | 기계공학 활용 |
|-----------|-----------|---------------|
| 기계 제작도 | 1/1 ~ 1/10 | 부품 가공, 공차 |
| 조립도 | 1/5 ~ 1/50 | 기어·베어링 배치 |
| 배치도 | 1/100 ~ 1/400 | 공장 레이아웃, 기계공작실 |
| 구조 도면 | 1/50 ~ 1/200 | 프레임 구조 해석 |
