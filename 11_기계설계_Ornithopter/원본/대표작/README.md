# 대표작 — Ornithopter 기계설계 (2024-2)

새의 날갯짓을 모사한 오니솝터 팀 설계 프로젝트의 기획 문서와 대표 어셈블리 모델입니다.

## 대표 파일

| 파일 | 종류 | 내용 |
|------|------|------|
| `Ornithopter_대표어셈블리.SLDASM` | CAD (SolidWorks) | 날개·기구 통합 대표 어셈블리 (Git LFS) |
| `chpt0-Team_project-birds_2024.pdf` | 문서 | 팀 프로젝트 기획/발표 자료 |
| `기소설계2_프로젝트(1).pdf` | 문서 | 기계설계 프로젝트 보고서 |

## 핵심 포인트

- 새의 날개·근육 구조를 분석해 **날갯짓(flapping) 기구로 추상화**한 생체 모방 설계.
- SolidWorks로 곡면 날개 파트(`wingpart.SLDPRT`)를 모델링하고 어셈블리 구속 부여.
- 다축 서보(5개) 순차 구동으로 날갯짓 동작 구현(상위 폴더 `#include Servo.h.txt`).
- 기획 문서 + 보고서 + 어셈블리 모델로 **설계 기획→모델링 흐름**을 제시.

> 구동 코드·설계 상세는 상위 분야 문서 참고 → [11. 기계설계 — Ornithopter](../../README.md)
