## 핵심 개념 정리

### 1. PM Servomotor 기본 수식

```
전기 방정식:
  V = R·i + L·(di/dt) + Kv·ω

토크 방정식:
  T = Kt·i

운동 방정식:
  J·(dω/dt) = T - B·ω - TL

  V  : 인가 전압 [V]
  R  : 전기자 저항 [Ω]
  L  : 인덕턴스 [H]
  i  : 전류 [A]
  Kv : 역기전력 상수 [V·s/rad]
  Kt : 토크 상수 [N·m/A]
  ω  : 각속도 [rad/s]
  J  : 관성 모멘트 [kg·m²]
  B  : 점성 감쇠 계수
  TL : 부하 토크
```

#### 예제 (PM Servomotor)

```
주어진 값:
  Kv = 0.824 V·s/rad
  Kt = 0.824 N·m/A  (SI 단위에서 Kv = Kt)
  R  = 1.5 Ω
  J  = 0.02 kg·m²

정격 속도에서 역기전력:
  Eb = Kv × ω

정격 전류:
  i = (V - Eb) / R

발생 토크:
  T = Kt × i
```

### 2. AC 모터 수식

#### 동기 전동기 (SM — Synchronous Motor)

```
동기 속도:
  Ns = 120·f / P   [rpm]
  ωs = 2πf / p     [rad/s]

  f : 전원 주파수 [Hz]
  P : 극수
  p : 극 쌍수 = P/2

토크:
  T = (3·V·Ef·sin δ) / (ωs·Xs)

  Ef : 여기 기전력
  Xs : 동기 리액턴스
  δ  : 부하각 (torque angle)
```

#### 유도 전동기 (IM — Induction Motor)

```
슬립(slip):
  s = (Ns - N) / Ns

회전수:
  N = Ns × (1 - s)   [rpm]

토크:
  T ∝ s·R2 / (R2² + s²·X2²)

  최대 토크 조건: s = R2 / X2
```

### 3. 기어 토크·속도 변환

```
기어비: G = N1/N2 = T2/T1 = ω1/ω2

  N1, N2 : 구동·피동 기어 잇수
  T1, T2 : 구동·피동 토크 [N·m]
  ω1, ω2 : 구동·피동 각속도 [rad/s]

등가 관성 (Gearing Effect):
  Je = J_motor + J_load / G²

등가 토크:
  Te = T_motor - TL / G
```

#### 주파수 응답 (Bode)

```
토크 상수 → 전달함수:
  G(s) = Kt / (J·s + B)

기어 포함:
  G_eq(s) = Kt / (Je·s + Be)

  → 기어비 G 증가 시 부하 관성 영향 G² 감소
  → 차단 주파수 (ωc) 상승 → 빠른 응답
```

### 4. Torsion (비틀림)

```
비틀림 응력:
  τ = T·c / J

  T : 비틀림 모멘트 [N·m]
  c : 단면 반지름 [m]
  J : 극관성모멘트 [m⁴]
    → 원형 단면: J = π·d⁴/32

비틀림각:
  φ = T·L / (G·J)

  L : 축 길이 [m]
  G : 전단탄성계수 [Pa]

기어 연결 축에서:
  T_input  × η = T_output
  φ_input  / G = φ_output × G  (역비례)
```
